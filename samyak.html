How the Timer Works

The timer is set in minutes. Once activated, the appliance powers ON immediately. When the countdown reaches zero, the relay cuts power — automatically.

Set any duration from 1 to 999 minutes
Appliance turns ON the moment the timer starts
Real-time countdown displayed on the interface
Automatic power-off when timer expires
Cancel at any time to cut power immediately
Toggle switch reflects real-time device state
Presets auto-fill duration and appliance
Lock panel with PIN for secure access

01

Plug In the Smart Switch

Insert the Smart Switch into your wall socket. Connect your appliance into the output socket.

02

Connect to Wi-Fi

On first use, the Smart Switch opens a setup hotspot. Enter your credentials and it joins your network.

03

Open the Control Panel

Access the built-in web interface from any browser. No app, no account. Just type the device IP.

04

Toggle Power or Set a Timer

Switch ON/OFF instantly, or set a countdown. When the timer expires, the relay cuts power automatically.

05

Appliance Powers Off Automatically

The relay disconnects the circuit safely. Energy saved, no manual intervention required.
